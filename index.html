Features

Summary

Generates summary for the serach both online and local device. This is mainly useful for the cops to know about any previous terriorist activity in breif without going through the new articles manually.

Multi-Lingual Summary

Includes the feature for generating summary in any local language, so that it can be used by all the cops accross the globe.

Re-Phrasing

The summarizer does not produce the sentences as it is. Here rephrasing apporach is used where the summary sentences are reframed without chnaging the actual context or mening of the sentences.

Word Count

The summarizer also provides an option to change the word count/sentnces in summary in a customized manner involving 5 or 10 or 20 senetences to be included in summary.
